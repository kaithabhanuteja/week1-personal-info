#  Personal Information Manager

## Project Description
This is my first Python project. It is a simple program that stores and displays personal information using variables, user input, and formatted output.

---

## Features
- Stores static information (name, age, city, hobby)
- Gets user input (favorite food and favorite color)
- Displays information in a clean formatted layout
- Calculates age in months
- Basic input validation for empty input
- Welcome and goodbye messages

---

## What I Learned
- **Variables:** Storing different types of data (string, integer)
- **Input/Output:** Taking user input and displaying results
- **String Formatting:** Using f-strings for clean output
- **Validation:** Handling empty user input
- **Basic Programming Structure**

---
